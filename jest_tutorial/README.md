# 요걸 어떻게 설정했나?

```sh
mkdir jest_tutorial
cd jest_tutorial
npm init
npm install --save-dev jest typescript
npm i -D ts-jest @types/jest
npx ts-jest config:init


# 무슨 설정 제거
# __test__
```
